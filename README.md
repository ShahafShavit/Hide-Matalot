# תוסף כרום: ניהול תצוגת מטלות בקורסים

## תאור הקוד

הקוד מאפשר למשתמש לנהל את תצוגת המטלות המופיעות בדף הראשי במודל, להגדיר התראות להגשה, ולהוסיף מטלה ליומן Google. המידע נשמר בצורה מקומית באמצעות IndexedDB ו-Storage של התוסף.

### פיצ'רים עיקריים:
1. **שימור מצב**: המטלות שהוסתרו נשמרות, כך שגם לאחר רענון הדף ההסתרה נשארת בתוקף.
2. **ניהול קל**: לחצן ניהול מאפשר למשתמש לבחור מטלות שיוסתרו או שיוצגו מחדש.
3. **התראות במחשב**: ניתן להגדיר התראה במחשב לפי מספר ימים ושעה לפני הדדליין. ההתראות רצות ברקע גם כשדף Moodle לא פתוח. בלחיצה על ההתראה, דף המטלה ייפתח.
4. **יומן Google**: ניתן להוסיף מטלה בודדת ליומן Google ישירות מתוך הטבלה.
5. **עדכון אוטומטי**: בעת שינוי מצב מטלה (הסתרה/הצגה), הקוד מעדכן את תצוגת התאריכים כך שתאריכים ללא מטלות לא יוצגו.
6. **ממשק משתמש ידידותי**: טבלה המאפשרת לסמן מטלות להסתרה, הוספת התראה למחשב והוספה ליומן גוגל בצורה נוחה.

---

## כיצד הקוד עובד

### שמירת והחזרת מצב
- **`getSavedState`**: פונקציה שמחזירה את מצב ההסתרה ששמור ב-IndexedDB.
- **`saveState`**: פונקציה ששומרת את המצב המעודכן ב-IndexedDB.
- **`cleanUpState`**: מנקה מידע על מטלות ישנות שאינן קיימות יותר בדף הנוכחי.

### טיפול בתצוגת תאריכים
- **`cleanUpDates`**: בודקת האם יש מטלות גלויות תחת כל תאריך. אם אין, התאריך מוסתר.

### זיהוי והצגת מטלות
- **`extractCourseExercisePairs`**: מזהה את המטלות בדף, כולל שם הקורס, שם המטלה, והקישור שלהן.
- **`displayTable`**: מציג טבלה שבה המשתמש יכול לבחור אילו מטלות להסתיר.
- **`addManagementButton`**: מוסיף כפתור ניהול המאפשר לפתוח את הטבלה לניהול תצוגת המטלות.

### התראות והפניות
- **`displayNotificationSettings`**: ממשק הגדרת התראות לכל מטלה לפי מספר ימים ושעה לפני הדדליין.
- **`background.js`**: מנהל התראות רקע, תזמון, ופתיחת קישור המטלה בלחיצה על ההתראה.
